ภาษาที่ใช้

HTML

*Html: ใช้ในการจัดโครงสร้างของหน้าเว็บไซต์ เช่น การใส่ข้อความหรือรูปภาพ และ การใส่ box ที่เข้าใส่ข้อความหรือการทำหน้าล้อกอิน (Login) ซึง Html จะทำหน้าที่หลักคือ แสดงหน้าเว็บไซต์

CSS

*Css: ใช้ในการจัดระเบียบให้สวยงานเพื่อให้ดูสวย เช่น การใส่สีหรือการจัดฟอร์มให้ดูเป็นระเบียบ และ จัดตำแหน่งในแต่ละช่องแต่ละบ็อก ซึง css จะทำหน้าที่หลักคือ จัดองค์ประกอบให้ดูสวยหรือเรียกอีกอย่างว่าตกแต่งหน้าเว็บ

ด้านออกแบบ

ออกแบบโพส์เตอร์

ในการออกแบบโพสเตอร์ คืองานที่ทำได้และอาจะดูไม่คอยตรงประเด่น แต่ถ้าทำแล้วก็ไม่หน่อยหน้าสำหรับมือใหญ่

ออกแบบหน้าเว็บไซต์

ออกแบบหร้าเว็บ แบบจัดตำแหน่งของโครงสร้างและตกแต่งให้กับหน้าเว็บ เช่น ใส่สี่ ทำให้เคลื่อนไว จัดระยะห่าง ใส่รูปภาพ ทำให้ใหญ่ จัดฟอร์น ตัวหน้า เช่น เว็บไซต์นี้ ใช่ภาษา Html กับ Css ในการจัดโครงสร้างและตำแหน่ง
